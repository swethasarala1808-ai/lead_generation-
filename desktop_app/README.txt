bizaxl LeadGen - Sales Intelligence Tool
=========================================
Replaces daily manual Google Maps search + Excel copy-paste.

HOW TO USE:
-----------
FIRST TIME SETUP:
  Double-click: install_and_run.bat
  (Installs Python dependencies automatically)

DAILY USE:
  Double-click: run.bat

HOW THE APP WORKS:
------------------
1. Select Industry (Medical, Retail, IT, etc.)
2. Type Area/Locality (e.g. "Kasturi Nagar", "Koramangala")
3. Select City
4. Set number of leads (10 to 150)
5. Click "Find Leads Now"
6. Results show: Name, Phone, Address, WhatsApp button
7. Click "Excel" to export — same format as your daily sheet

DATA SOURCE:
------------
Uses OpenStreetMap (free, no cost, no API key needed)
Works exactly like Google Maps "near me" search
