====================================================
  bizaxl LeadGen - Sales Intelligence Tool
  Automates daily lead collection for sales team
====================================================

WHAT THIS APP DOES:
-------------------
Replaces manual Google Maps search + Excel copy-paste.
Your marketing team used to:
  1. Search Google Maps ("medical shops near KR Puram")
  2. Copy each business: Name, Phone, Address
  3. Paste into Excel
  4. Send to sales team

Now this app does all of that automatically:
  1. Select industry (Medical, Retail, IT, CA firms...)
  2. Type area (Kasturi Nagar, Koramangala, Andheri...)
  3. Select city
  4. Click "Find Leads Now"
  5. Get results with Name, Phone, Address, WhatsApp
  6. Export to Excel in one click


HOW TO RUN (First Time):
------------------------
Option A - Double-click installer:
  → Double-click INSTALL_AND_RUN.bat
  → It installs everything and starts the app

Option B - Manual:
  1. Install Python: https://python.org (check "Add to PATH")
  2. Open Command Prompt
  3. cd to this folder
  4. pip install PyQt5 requests openpyxl
  5. python app.py

Option C - Build .exe (run once):
  → Double-click build_exe.bat
  → Creates dist/bizaxl_LeadGen.exe
  → Share that .exe with your team


HOW TO USE:
-----------
1. Select industry on the left (Medical Shops, Retail, CA firms etc.)
2. Type your area: "Kasturi Nagar" or "KR Puram" or "Koramangala"
3. Select city: Bangalore, Mumbai, Delhi etc.
4. Move slider to choose how many leads (10-150)
5. Click "Find Leads Now"
6. Results show: Business Name, Phone, Address
7. Click phone number to call directly
8. Click WhatsApp to message directly
9. Click "Excel" to download the list


WHAT DATA YOU GET:
------------------
- Business Name
- Phone Number (click to call)
- WhatsApp button (pre-written intro message)
- Full Address with Pincode
- Email (when available)
- Website (when available)
- Opening Hours (when available)
- Google Maps link
- Quality Score

Data Source: OpenStreetMap (100% Free, No API Key)
